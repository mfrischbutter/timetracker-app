
# Timetracker App

The app version of the Mogic time tracker.

## How to install
Android

    flutter build apk --release

iOS (not tested yet)
...
## Developement
Clone repository

    git clone git@github.com:mfrischbutter/timetracker-app.git
    cd ./timetracker-app

Install all flutter packages

    flutter pub get

Run app on connected device or emulator

    flutter run
